# Sagooda 🔍

Smart Recon & Analysis Tool for Ethical Hacking

## Features

* Fast port scanning
* Full port scanning (optional)
* Service detection
* Banner grabbing
* Subnet scanning

## Usage

```bash
sagooda -t 127.0.0.1
sagooda -t 192.168.1.1
sagooda -t 192.168.1.0/24
```

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/sagooda.git
cd sagooda
chmod +x sagooda
sudo mv sagooda /usr/local/bin/
```
